About this app

Instructions for using the app

  • Condition 1: Arrow keys(implicit traversal) to navigate through the buttons and enter key to select the highlighted button
  • Condition 2: "A" key to move left and "D" key to move right in the interface. Enter to select the highlighted button

Test Description

This interface was tested between the two conditions listed above. The following equations were used for testing

  1. (483 / 19) * (25) + (60) - 7
  2. (650 + 17) * (286) / (3) - 9
  3. (4 * 5) * (10) + (738) - (62) / 9
  4. (3 / 7) * (25) - (89) + (46) * 10
  5. (82) * (40) / (713) * (96) + 5
