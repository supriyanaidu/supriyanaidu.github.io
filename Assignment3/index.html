Accessibility features

  • Every button is accessible via the tab key
  • Header tabs and dialer keys have keyboard shortcuts
  • Formatted contact table for ease of use via a screen reader
  • All images are captioned
  • ARIA roles are present
  • Dynamic tab change has ARIA-live
